# Images produits Scout

## Convention de nommage
`{id_produit}.jpg` ou `{id_produit}.png` ou `{id_produit}.webp`

L'ID doit correspondre exactement à l'ID en base de données.

## Exemples
- `bayraktar_tb2.jpg` → Bayraktar TB2
- `mq9_reaper.jpg` → MQ-9 Reaper  
- `dji_mavic3e.jpg` → DJI Mavic 3 Enterprise
- `elistair_orion2.jpg` → Elistair Orion 2
- `milrem_themis.jpg` → THeMIS UGV
- `sea_baby.jpg` → Sea Baby USV

## Format recommandé
- JPG ou WebP
- 800×500px minimum (ratio 16:10)
- < 500KB par image

## Upload
Glisse les images ici et commit+push.
Le site les charge automatiquement via GitHub Pages.
